## Building

**Requirements**

* C++11 compiler
* CMake 3.6 or later
* Qt 5.6 (Might also work in earlier Qt5 versions - not tested)

**How to build:**

      > git clone https://github.com/jahnf/projecteur
      > cd projecteur
      > mkdir build && cd build
      > QTDIR=/opt/Qt/5.9.6/gcc_64 cmake ..
      > make
